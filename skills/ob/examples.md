# OB 技能示例

## 示例 1：输出内容构思

### 输入

```
现在我来构思我的输出内容我以后输出的内容都不会有三个模块的内容：第一个模块是生活中的痛点，第二个模块是解决痛点的方法，第三个模块是解决办法的安装过程或者说是操作过程输出的内容会有三个层级：第一个成绩是纯文字输出主要在 Obsidian 中完成第二个成绩是图文并茂的输出也就是文字加上操作办法的截图第三个层级是视频输出诶
```

### 输出

```markdown
# 输出内容构思

现在我来构思我的输出内容。

输出的内容都会有 3 个模块：

1. 生活中的痛点
2. 解决痛点的方法
3. 解决办法的安装过程或操作过程

输出的内容会有 3 个层级：

**第一层级：** 纯文字输出，主要在 Obsidian 中完成
**第二层级：** 图文并茂的输出，文字加上图片
**第三层级：** 视频输出
```

## 示例 2：深夜思考

### 输入

```
这是我工作到半夜躺在床上睡觉之前的思考我想到了两个并线的路径然后是可以测试的几乎零成本的 MVP 第一个就是利用 Geo 销售螺蛳粉我自己可以拿到低价的螺蛳粉并且可以一键发货那么我就可以调查一下如果用户想在 AI 搜索中搜索螺蛳粉他会从哪里得到什么信息那么我就从信息来源或者说是到信息员去发布高权重的笔记或者信息让所有收螺蛳粉的人第一个就收到我其中要植入我的联系方式或者说是销售渠道这整个过程就需要不断地进行测试而且是一个长期的测试那么这就需要一个计划了这个计划必须包括寻找用户是怎么收螺蛳粉的是否会有需要求购螺蛳粉的用户在 AI 上进行搜索他们是具体是怎么搜索的要在每一个哎呀入口进行了一个测试然后收集 AI 给出的信息源到底是在哪里然后再研究这个信息源的内容需要哪一些还需要研究如何取得这个高权重的内容主要是是什么结构什么形式的最后还得研究如何巧妙地将他引揉到我的销售网络引流完之后如何付款以及下单最后是和我的发货人如何进行这个最后的发货以及售后而且整个过程有哪些可以直接流程化几乎不需要人为地去操作
第二个同理我现在微信上加了非常多的零七食品销售人员的微信他们之中有售卖人去食品的也有收购人气食品的我就需要一个程序自动的把他们的朋友圈内容还有发到我微信上的内容进行搜索总结然后找到其中的商机这个想法还是不太成熟但是我既然有这些人的资源我就一定可以干一些事情一些能赚钱的事情
```

### 输出

```markdown
# 深夜思考：两个低成本的变现业务路径

这是我工作到半夜躺在床上睡觉之前的思考。我想到了两个变现的路径，都是几乎零成本可测试的 MVP。

## 路径一：利用 AI 搜索销售螺蛳粉

利用 Geo 销售螺蛳粉。我自己可以拿到低价的螺蛳粉，并且可以一件发货。

调查一下如果用户想在 AI 搜索中搜索螺蛳粉，他会从哪里得到什么信息。从信息来源处发布高权重的笔记或信息，让所有搜索螺蛳粉的人第一时间搜索到我。

其中要植入我的联系方式或销售渠道。这整个过程需要不断测试，且是长期测试。

因此需要一个计划，该计划必须包括以下内容：

1. 寻找用户是怎么搜索螺蛳粉的
2. 是否会有需要求购螺蛳粉的用户在 AI 上进行搜索
3. 他们具体是怎么搜索的，要在每个 AI 入口进行测试
4. 收集 AI 给出的信息源到底是在哪里
5. 研究这个信息源的内容需是哪一些
6. 研究如何取得这个高权重的内容，主要是结构和形式
7. 研究如何巧妙地将他引入到我的销售网络
8. 引流完之后如何付款以及下单
9. 和我的发货人如何进行最后的发货以及售后
10. 整个过程有哪些可以直接流程化，几乎不需要人为操作

## 路径二：整合微信临期食品销售人员资源

我现在微信上加了非常多的临期食品销售人员的微信，其中有售卖食品的，也有收购食品的。

我需要一个程序自动把他们的朋友圈内容以及发到我微信上的内容进行搜索总结，从而找到其中的商机。

这个想法虽然还不太成熟，但我既然有这些人的资源，就一定可以干一些能赚钱的事情。
```

## 示例 3：会议记录

### 输入

```
嗯今天我们开了一个产品评审会啊然后主要讨论了三个事情第一个是关于我们新功能的上线时间嗯原本计划是下周但是测试发现了一些 bug 所以需要延期然后第二个事情是关于用户反馈的我们的客服团队收集了很多用户意见主要是说界面太复杂了需要简化第三个是下季度的规划我们要开始做移动端了嗯大概就是这样

```

### 输出

```markdown
# 产品评审会会议记录

今天我们开了一个产品评审会。主要讨论了三个事情：

## 1. 新功能上线时间

原本计划是下周，但是测试发现了一些 bug，所以需要延期。

## 2. 用户反馈

我们的客服团队收集了很多用户意见，主要是说界面太复杂了，需要简化。

## 3. 下季度规划

我们要开始做移动端了。
```
